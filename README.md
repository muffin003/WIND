## üöÄ 3. –ó–∞–ø—É—Å–∫ –°—Ü–µ–Ω–∞—Ä–∏–µ–≤

### –°—Ü–µ–Ω–∞—Ä–∏–π A: –¢–æ–ø–æ–ª–æ–≥–∏—è –£—Å—Ç–æ–π—á–∏–≤–æ—Å—Ç–∏ (Mapping Landscape)
**–¶–µ–ª—å:** –ü–æ—Å—Ç—Ä–æ–∏—Ç—å 3D-–ø–æ–≤–µ—Ä—Ö–Ω–æ—Å—Ç—å –∏ —Ç–µ–ø–ª–æ–≤—É—é –∫–∞—Ä—Ç—É, –ø–æ–∫–∞–∑—ã–≤–∞—é—â—É—é, –ø—Ä–∏ –∫–∞–∫–∏—Ö —Å–æ—á–µ—Ç–∞–Ω–∏—è—Ö –≥–ª–∞–¥–∫–æ—Å—Ç–∏ ($\rho$) –∏ —Å–∫–æ—Ä–æ—Å—Ç–∏ –¥—Ä–µ–π—Ñ–∞ ($A$) –∞–ª–≥–æ—Ä–∏—Ç–º —Å–ø–æ—Å–æ–±–µ–Ω —É–¥–µ—Ä–∂–∏–≤–∞—Ç—å —Ü–µ–ª—å.

```python

rho_grid = [0.2, 0.6, 1.0]       # –ú–æ–∂–Ω–æ —Å–¥–µ–ª–∞—Ç—å –≥—É—â–µ: np.arange(0.1, 1.1, 0.1)
drift_grid = [0.0, 0.05, 0.10]

results = []

for d in drift_grid:
    for r in rho_grid:
        # –ö–æ–Ω—Ñ–∏–≥—É—Ä–∞—Ü–∏—è –¥–ª—è —Ç–µ–∫—É—â–µ–π —Ç–æ—á–∫–∏ —Å–µ—Ç–∫–∏
        cfg = {
            'dim': 2, 
            'rho': r, 
            'drift_speed': d, 
            'noise_type': 'gaussian', 
            'noise_scale': 0.5, 
            'geometry': 'ideal'
        }
        
        # 1. –ê–≤—Ç–æ-—Ç—é–Ω–∏–Ω–≥ (–ö—Ä–∏—Ç–∏—á–µ—Å–∫–∏ –≤–∞–∂–Ω–æ –ø–µ—Ä–µ–Ω–∞—Å—Ç—Ä–∞–∏–≤–∞—Ç—å –¥–ª—è –∫–∞–∂–¥–æ–≥–æ rho!)
        best_hp = tuner.tune(cfg)
        
        # 2. –ó–∞–ø—É—Å–∫ (min_runs –º–æ–∂–Ω–æ —Å–Ω–∏–∑–∏—Ç—å –¥–ª—è —Å–∫–æ—Ä–æ—Å—Ç–∏ –ø–æ—Å—Ç—Ä–æ–µ–Ω–∏—è –∫–∞—Ä—Ç—ã)
        errors, success, traj = run_adaptive_experiment(
            MyOptimizer, best_hp, cfg, min_runs=30, tol=0.15
        )
        
        results.append({
            'type': 'heatmap', 'rho': r, 'drift': d, 
            'success': success, 'errors': errors
        })
```

### –°—Ü–µ–Ω–∞—Ä–∏–π B: –°—Ç—Ä–µ—Å—Å-—Ç–µ—Å—Ç "–¢—è–∂–µ–ª—ã–µ —Ö–≤–æ—Å—Ç—ã" (Pareto Noise)

    'noise_type': 'pareto',  # <--- –í–ö–õ–Æ–ß–ï–ù–ò–ï –¢–Ø–ñ–ï–õ–´–• –•–í–û–°–¢–û–í
    'noise_scale': 1.0,      # –í—ã—Å–æ–∫–∏–π —É—Ä–æ–≤–µ–Ω—å —à—É–º–∞
    'geometry': 'ideal'
}

# –¢—Ä–µ–±—É–µ—Ç —Å—Ç—Ä–æ–≥–æ–π —Å—Ç–∞—Ç–∏—Å—Ç–∏–∫–∏ (tol=0.05), —Ç–∞–∫ –∫–∞–∫ –¥–∏—Å–ø–µ—Ä—Å–∏—è –æ–≥—Ä–æ–º–Ω–∞—è

## üì• 4. –ö–∞–∫ –ø–æ–¥–∫–ª—é—á–∏—Ç—å —Å–≤–æ–π –∞–ª–≥–æ—Ä–∏—Ç–º

–ß—Ç–æ–±—ã –ø—Ä–æ—Ç–µ—Å—Ç–∏—Ä–æ–≤–∞—Ç—å –≤–∞—à —Å–æ–±—Å—Ç–≤–µ–Ω–Ω—ã–π –º–µ—Ç–æ–¥, —Å–æ–∑–¥–∞–π—Ç–µ –∫–ª–∞—Å—Å, –Ω–∞—Å–ª–µ–¥—É–µ–º—ã–π –æ—Ç `BaseOptimizer`.

```python
class MyCustomOptimizer(BaseOptimizer):
    def step(self, x, oracle_data):
        """
        x: np.array - –¢–µ–∫—É—â–∏–µ –∫–æ–æ—Ä–¥–∏–Ω–∞—Ç—ã
        oracle_data: dict - {'value': float, 'grad': np.array}
        """
        
        # 1. –†–∞—Å–ø–∞–∫–æ–≤–∫–∞ –¥–∞–Ω–Ω—ã—Ö
        grad = oracle_data.get('grad')
        
        # 2. –î–æ—Å—Ç—É–ø –∫ –≥–∏–ø–µ—Ä–ø–∞—Ä–∞–º–µ—Ç—Ä–∞–º (–æ–ø—Ä–µ–¥–µ–ª—è—é—Ç—Å—è –≤ tuner)
        lr = self.hp.get('lr', 0.01)
        
        # 3. –í–∞—à–∞ –ª–æ–≥–∏–∫–∞ –æ–±–Ω–æ–≤–ª–µ–Ω–∏—è
        # (–ù–∞–ø—Ä–∏–º–µ—Ä, –ø—Ä–æ—Å—Ç–æ–π –≥—Ä–∞–¥–∏–µ–Ω—Ç–Ω—ã–π —Å–ø—É—Å–∫)
        x_new = x - lr * grad
        
        return x_new
```

## üìä 5. –ß—Ç–µ–Ω–∏–µ –æ—Ç—á–µ—Ç–æ–≤ (Visualizer)

–ö–ª–∞—Å—Å `BenchmarkVisualizer` –≥–µ–Ω–µ—Ä–∏—Ä—É–µ—Ç –¥–∞—à–±–æ—Ä–¥. –ö–∞–∫ —á–∏—Ç–∞—Ç—å –≥—Ä–∞—Ñ–∏–∫–∏:

1.  **3D Error Topology:** –ï—Å–ª–∏ –ø–æ–≤–µ—Ä—Ö–Ω–æ—Å—Ç—å —Ä–µ–∑–∫–æ —É—Ö–æ–¥–∏—Ç –≤–≤–µ—Ä—Ö –≤ –ª–µ–≤–æ–º –≤–µ—Ä—Ö–Ω–µ–º —É–≥–ª—É (–º–∞–ª–æ–µ $\rho$, –≤—ã—Å–æ–∫–∏–π –¥—Ä–µ–π—Ñ) ‚Äî —ç—Ç–æ –Ω–æ—Ä–º–∞–ª—å–Ω–æ. –ï—Å–ª–∏ –æ–Ω–∞ —Ä–∞—Å—Ç–µ—Ç –≤–µ–∑–¥–µ ‚Äî –∞–ª–≥–æ—Ä–∏—Ç–º –ø–ª–æ—Ö.
2.  **Robustness Radar:** –ß–µ–º —à–∏—Ä–µ –ø–ª–æ—â–∞–¥—å –º–Ω–æ–≥–æ—É–≥–æ–ª—å–Ω–∏–∫–∞, —Ç–µ–º —É–Ω–∏–≤–µ—Ä—Å–∞–ª—å–Ω–µ–µ –∞–ª–≥–æ—Ä–∏—Ç–º.
